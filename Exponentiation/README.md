## Fast Exponentiation : a^b
> Time Complexity O(log b)

## Modular Arithmetic Properties:
    * (a+b)%c = ((a%c)+(b%c))%c
    * (a-b)%c = ((a%c)-(b%c)+c)%c
    * (a*b)%c = ((a%c)*(b%c))%c
    * (a/b)%c = ((a%c)/(b%c))%c

## Modular Exponentiation : (a^b)%c = ((a%c)^(b%c))%c
>Time Complexity O(log b)
